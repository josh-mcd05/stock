Run the jar file by navigating to the file location in a terminal and running 
"java -jar Homework4.jar"

Once in the program, you will start will a portfolio that contains 3 stocks: google, amazon,
And Nvidia. To add more, just type "add stock-ticker" and click the enter button.

To create a second portfolio, type "create-portfolio" and hit the enter button.
This will create a second empty portfolio and set it as the active portfolio. To add stocks,
Just type "add stock-ticker" as many times as you would like.

To get the value of the current portfolio on a given date, type "value YYYY-MM-DD". All dates entered should be in YYYY-MM-DD format. You can switch between portfolios by using the command
"set-portfolio number", the first portfolio is numbered 1 and so on.
